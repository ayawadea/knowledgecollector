lib/firebase.ts 
import { initializeApp } from "firebase/app";
import { getFirestore } from "firebase/firestore";
import { getStorage } from "firebase/storage";

const firebaseConfig = {
  apiKey: process.env.NEXT_PUBLIC_FIREBASE_API_KEY,
  authDomain: process.env.NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN,
  projectId: process.env.NEXT_PUBLIC_FIREBASE_PROJECT_ID,
  storageBucket: process.env.NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET,
};

const app = initializeApp(firebaseConfig);
export const db = getFirestore(app);
export const storage = getStorage(app);

Setup Guide

1

Initialize Firebase

Create a project in Firebase Console. Enable Firestore, Storage, and Authentication modules.

2

Environment Variables

Create a .env.local file and populate it with your Firebase Web App configuration.

3

Install Dependencies

Run npm install to fetch the core framework and Tiptap extensions.

4

Launch Dev Server

Execute npm run dev and navigate to localhost:3000.

UI/UX Design Review

Color Palette

Optimized for high-contrast readability and reduced eye strain during long writing sessions.

Design Principles

Typography

Inter (Sans Serif) for UI, 1.25 modular scale for content hierarchy. 16px base ensures legibility across all devices.

Responsiveness

Three-pane layout for desktop productivity; fluid collapse to single-pane with intuitive drawers for mobile.

Motion

Framer Motion handles subtle transitions, theme switching, and haptic-like hover feedback.

Accessibility

WCAG 2.1 AA compliant. Keyboard-first navigation and full ARIA support for screen readers.

Integrated Experience

More than just a notebook. Knowledge Collector integrates your "Today's Focus" directly into your workspace sidebar. Stay organized with real-time task management and a persistent clock widget.
